🇰🇷 소개 / Overview (Korean)

Answer Grid는 문제 풀이 연습을 위해 만든 OMR 스타일의 컴팩트 답안지 생성 도구입니다.
책에 필기하는 것을 선호하지 않기에 제작하게 되었습니다. 

이 도구는 원하는 문항 수(10~100문항)를 즉시 생성할 수 있으며,
번호 동그라미 표시, 외곽선 표시 여부를 선택할 수 있고,
PDF 저장 기능을 통해 아이패드·모바일·PC에서 모두 사용할 수 있습니다.

간단한 UI와 가벼운 구조로 구성되어 있어,
개인용 연습·자격증 공부·모의고사 기록 용도로 활용하기 좋습니다.
프론트엔드 기초 스택(HTML/CSS/JS)을 기반으로 구현했으며,
DOM 동적 생성, 프린트 스타일 분리(print.css), 컴팩트 그리드 렌더링 등이 핵심 기술 포인트입니다.

🇯🇵 説明 / Overview (Japanese)

Answer Grid（アンサーグリッド） は、練習用の OMR風コンパクト解答用紙 を生成するツールです。
本に直接書き込みたくないため、iPad やノートに答えを写しながら勉強していましたが、
既存のオンライン答え表は 五択向けのフォーマットしかなく、不便に感じたため自作しました。

問題数（10〜100問）を即座に生成でき、
番号の丸表示・外枠表示の ON/OFF、PDF 保存など、
モバイル・iPad・PC どれでも使いやすいように設計しています。

軽量な UI とシンプルな構成で、
資格試験の勉強、模擬試験の記録、個人学習などに最適です。
HTML/CSS/JavaScript を中心に構築しており、
DOM の動的生成、print.css による印刷レイアウト分離、
コンパクトなグリッド描画などが技術的なポイントです。
